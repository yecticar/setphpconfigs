# setphpconfigs

This project is intended to help a user set the homebrew/macports php config files with the desired values
- Experimental for custom installation

# Requirements

- GNU Utils (with the `g` prefix)
  - greadlink
  - gsed

# Configuration

Default values are set in the `config.sh` file, if you want to override them just copy the file

```bash
cp config.sh config.local.sh
```

and modify the `config.local.sh` file with your own values
